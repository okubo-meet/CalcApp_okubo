# はじめてのけいさん - 幼児向け電卓アプリ

## 1.概要
このアプリは3歳くらいの子どもを対象にした知育アプリです。

可愛い見た目や、タップ音で興味を引き、計算結果を'AVSpeechSynthesizer'を使用して読み上げることで、子どもが目と耳で算数を楽しむことができます。
